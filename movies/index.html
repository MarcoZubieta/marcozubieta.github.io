Videotutoriales del Fablab

Una biblioteca viva para documentar procesos: impresión 3D, corte láser, electrónica, IA aplicada, diseño paramétrico, automatización y prototipos. La idea es simple: cada video deja una huella útil para que otros puedan repetir, mejorar y remixar.

Para videos livianos puedes usar archivos locales en /movies/media/. Para cursos largos, lo más robusto es subir el video a YouTube, Vimeo, Cloudflare Stream u otro alojamiento de video y embeberlo aquí. GitHub Pages es excelente para la web estática; no es ideal como bodega pesada de video.

Cómo subir un nuevo videotutorial

Opción local, para archivos pequeños:

movies/
  index.html
  videos.js
  media/
    mi-videotutorial.mp4
  posters/
    mi-videotutorial.jpg

Luego editas movies/videos.js y agregas una entrada nueva al catálogo.
